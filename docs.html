Comment obtenir une clé API Google AI (Gemini)

Mis à jour le 21 Novembre 2025 • Temps de lecture : 2 min

Pet utilise la puissance de l'intelligence artificielle de Google (Gemini) pour analyser vos notes et générer automatiquement des quiz et des flashcards pertinents. Pour que cela fonctionne, vous avez besoin d'une clé API gratuite.

Suivez ces étapes simples pour obtenir votre clé en moins de 2 minutes.

Étape 1 : Accéder à Google AI Studio

Rendez-vous sur le site officiel de Google pour les développeurs IA : Google AI Studio.

Si vous n'êtes pas déjà connecté, connectez-vous avec votre compte Google habituel.

Étape 2 : Créer une clé API

Une fois sur le tableau de bord :

  1. Cliquez sur le bouton "Get API key" situé dans le menu de gauche (icône de clé).
  2. Cliquez sur le bouton bleu "Create API key".
  3. Si demandé, choisissez un projet existant ou sélectionnez "Create API key in new project".

Étape 3 : Copier la clé

Une fenêtre va s'ouvrir affichant votre nouvelle clé API. Elle ressemble à une longue suite de caractères commençant par AIza....

Cliquez sur le bouton de copie à côté de la clé pour la copier dans votre presse-papiers.

Important : Ne partagez jamais cette clé avec personne. Elle est liée à votre compte Google.

Étape 4 : Configurer Pet

Maintenant que vous avez votre clé :

  1. Ouvrez l'application Pet.
  2. Cliquez sur l'icône de Paramètres (roue dentée) en bas à gauche.
  3. Dans la section "Intelligence Artificielle", collez votre clé dans le champ "Clé API Google Gemini".
  4. L'application enregistrera automatiquement votre clé.

C'est tout ! Vous pouvez maintenant importer vos cours et laisser l'IA générer du contenu de révision pour vous.
